# linux install zsh

## download repository

```.env
git clone https://github.com/loveagri/zsh.git

cd zsh
```

## install zsh

```.env
# centos 
sudo sh centos.sh

# ubuntu
sudo sh ubuntu.sh

# mac
sudo sh mac.sh
```

## install ohmyzsh and  incr
```.env
sh ohmyzsh.sh
```

## load configuration
```.env
source $HOME/.zshrc
source $HOME/.incr-02.zsh
```
